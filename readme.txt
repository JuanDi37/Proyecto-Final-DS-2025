Este proyecto utiliza datos de la encuesta BRFSS 2015 del CDC para desarrollar un modelo de clasificación binaria que predice si una persona tiene o podría tener diabetes, a partir de variables relacionadas con salud y estilo de vida. El dataset contiene 253,680 observaciones y 22 variables, de las cuales al menos 18 son categóricas. Originalmente, la variable Diabetes_012 incluye tres clases: 0 para personas sin diabetes, 1 para prediabetes y 2 para diabetes. Para cumplir con el enfoque de clasificación binaria, se utiliza la variable derivada Diabetes_binary, donde 0 indica no diabetes y 1 agrupa tanto prediabetes (1) como diabetes (2).
